# Sorvetes Pakila 🇺🇸
Developing a website for my grandfather's ice cream shop, located in Varre-Sai, RJ.

In this project I am practicing my skills in HTML, CSS and JavaScript,
focusing on responsive design and clean code structure.

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Responsive layout (mobile and desktop)
- Flavor menu (ice creams and popsicles)
- WhatsApp contact button
- Store location and opening hours

*using generic emojis to represent some flavours for now, i'm working on getting original flavours images directly from the factory.
